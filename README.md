# SSNLC Law College Website

This is the official website for SSNLC Law College.

## Project Structure

- `client/` - Next.js frontend application
- `server/` - Express.js backend API

## Installation

After cloning this repository, you need to install the dependencies:

### Client

```bash
cd client
npm install
```

### Server

```bash
cd server
npm install
```

## Development

### Client

```bash
cd client
npm run dev
```

### Server

```bash
cd server
npm run dev
```

## Building for Production

### Client

```bash
cd client
npm run build
npm start
```

### Server

```bash
cd server
npm start
```
